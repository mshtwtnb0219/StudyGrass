## 説明  
個人学習した科目・時間を記録してheatmap及びdountGrphに表示する  
更新画面がないため何れ作りたい(=ﾟωﾟ)ﾉ  
登録データはfirebaseのfireStoreを利用

6段階に分けてheatmapの色付けする時間が多いほど色が濃くなる  
1分～30分  
31分～60分  
61分～90分  
91分～120分  
121分～150分  
151分～180以上
